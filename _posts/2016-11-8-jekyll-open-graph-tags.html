If you're building a site using Jekyll, then the following code snippet may be just what you need for implementing Open Graph tags on your site. The code snippet below can be put in it's own partial, for easy inclusion in every Jekyll project that you undertake. Below the snippet, I have outlined the variables used, with code snippets related to the _config.yml file as well as the YAML front matter for individual pages & posts.

Tags & Variables

Please use the following table as a reference when adding your site and/or page variables in Jekyll.

Tag Page Variable Fallback Site Variable Default
<title> page.title site.title N/A
description page.excerpt site.description N/A
og:site_name N/A site.title N/A
og:determiner page.determiner N/A N/A
og:title page.title site.title N/A
og:description page.excerpt site.description N/A
og:type page.opengraph_type site.opengraph_type website
og:url N/A N/A Page Url
og:locale(:alternate) N/A site.locales (Array) N/A
fb:app_id N/A site.facebook.app_id N/A
twitter:card N/A N/A summary
twitter:site N/A site.author.twitter N/A
twitter:creator page.author.twitter site.author.twitter N/A
rel="author" page.author.google site.author.google N/A

The Open Graph image tag implentation is not really describable in a table, however the implementation is fairly simple to work out.

The code also includes an optional Facebook Pixel script, which will also be included should the site.facebook.pixel_id be provided.

Config.yml

An example config.yml file with just the necessary parameters is shown below.

{% highlight yaml %}

title: Code Theorist

email: codetheorist@gmail.com

description: > 
  The brand new website from Code Theorist, built using Jekyll.

author:
  name: Code Theorist
  email: codetheorist@gmail.com
  google: '116052050658604872041'
  twitter: 'codetheorist'

image:
  src: '/image.jpg'
  height: 726
  width: 1366
  type: image/jpeg

facebook:
  app_id: '1234567890123456'
  pixel_id: '1234567890123456'

locales:
  - 'en-GB'
  - 'en-US'

{% endhighlight %}
Front Matter

The following is some sample front matter containing all of the necessary required page variables.

{% highlight yaml %}
---
title: Code Theorist
excerpt: 'If this is ommitted, the page excerpt can be automatically generated from the page content'
determiner: 'the'
opengraph_type: 'article'
author:
  google: '116052050658604872041'
  twitter: 'codetheorist'
image:
  src: '/image.jpg'
  height: 726
  width: 1366
  type: image/jpeg

{% endhighlight %}
